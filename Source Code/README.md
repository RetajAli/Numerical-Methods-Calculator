# Source Code

This folder contains the complete source code of the Numerical Methods Calculator web application.

The application was developed using HTML, CSS, and JavaScript to provide an interactive environment for solving numerical analysis problems.

Implemented Methods:

## Root Finding Methods

* Bisection Method
* Secant Method
* Fixed Point Iteration
* Newton-Raphson Method

## Linear Systems

* Jacobi Method
* Gauss-Seidel Method

## Interpolation Methods

* Lagrange Interpolation
* Newton Forward Finite Differences
* Newton Backward Finite Differences
* Newton Forward Divided Differences
* Newton Backward Divided Differences

## Numerical Differentiation

* Three-Point Formula
* Five-Point Formula

The source code includes user input validation, dynamic result generation, iteration tables, and mathematical computations for each implemented numerical method.
