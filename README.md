# Numerical Methods Calculator

## Overview

Numerical Methods Calculator is a web-based application developed to solve various numerical analysis problems through an interactive and user-friendly interface.

The project provides implementations of several numerical methods commonly used in engineering, mathematics, and computer science courses. Users can enter input values and instantly obtain calculated results along with iteration details.

---

## Features

### Root Finding Methods

* Bisection Method
* Secant Method
* Fixed Point Iteration
* Newton-Raphson Method

### Linear System Solvers

* Jacobi Method
* Gauss-Seidel Method

### Interpolation Methods

* Lagrange Interpolation
* Newton Forward Finite Differences
* Newton Backward Finite Differences
* Newton Forward Divided Differences
* Newton Backward Divided Differences

### Numerical Differentiation

* Three-Point Formula
* Five-Point Formula

### Additional Features

* Interactive User Interface
* Dynamic Iteration Tables
* Input Validation
* Instant Result Calculation
* Mathematical Expression Evaluation
